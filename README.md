# AeroView

Projet de surveillance de la qualité de l'air en temps réel.
Le but du site est de visualiser la pollution (AQI, PM2.5, Ozone...) partout dans le monde et de donner des conseils adaptés (sportifs, personnes sensibles).

## Fonctionnalités
* Carte interactive : Visualisation de la pollution sur une carte mondiale.
* Recherche : Barre de recherche pour trouver n'importe quelle ville.
* Données détaillées : Affichage du NO2, O3, PM10 et PM2.5.
* Comparateur : Permet de comparer la qualité de l'air entre deux villes.
* Profils : Mode "Sportif" ou "Sensible" pour filtrer les conseils santé.
* Interface : Mode Sombre/Clair et affichage responsive (adapté mobile).

---

## Technologies utilisées
* React.js (Frontend)
* Open-Meteo API (Données qualité de l'air)
* Nominatim / OpenStreetMap (Géolocalisation)
* Leaflet (Cartes)
* Recharts (Graphiques)

---

## Lancer le projet

Pour tester le site en local :

1. Cloner le repo :
   git clone https://github.com/votre-pseudo/aeroview.git

2. Installer les librairies :
   npm install

3. Lancer le serveur :
   npm start

L'application sera accessible sur http://localhost:3000 dans le navigateur.
